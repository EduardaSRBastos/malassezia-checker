## Features

- **Ingredient Analysis**: Automatically identifies and highlights problematic ingredients.
- **Product Search**: Browse by brand and product to quickly load ingredient lists.
- **Manual Input**: Paste your own ingredient list for instant evaluation.
- **Severity Ratings**: Categorized flags (red, orange, yellow) to help assess reactivity risk.

<br>

## How to Use

1. **Choose a Brand** from the dropdown list.
2. **Select a Product** or paste ingredients manually.
3. **Click "Check"** to analyze the ingredients.
4. **Review the Results**, including severity color ratings and ingredient categories.
